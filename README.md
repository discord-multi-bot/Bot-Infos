# Server Defender Bot

**Server Defender Bot** ist ein Discord-Sicherheitsbot, der deinen Server vor Spam, Raids und Missbrauch schützt. Mit automatischen Moderationsfunktionen, Log-Systemen und Anti-Raid-Technologien bleibt dein Server sicher.

---

## Features
- **Anti-Raid & Anti-Spam:** Schutz vor Massenangriffen und Spammern  
- **Automatische Moderation:** Kicks, Bans und Rollen-Management  
- **Logging & Benachrichtigungen:** Übersicht über alle Aktionen  
- **Einfache Konfiguration:** Per Bot-Befehle steuerbar  
- **DSGVO-freundlich:** Speichert nur notwendige Daten
